K-means(K is given)

  1. Partition all data points into K subsets(non empty)
  1. Select a seed(data point) from each subset as centroid.(K centroids)
  1. Partition the rest of data points using the centroids ← reassignment of data points
    1. For each data point, assign it to the cluster where it has the closest distance to the centroid of the cluster
  1. Recalculate the centroids for each cluster
    1. For each cluster, take the mean result of all the data points as the new centroids(may not be a data point)
  1. Go to step 3. to reassign data points
    1. Keep doing these steps until no new reassignment of data points happens
