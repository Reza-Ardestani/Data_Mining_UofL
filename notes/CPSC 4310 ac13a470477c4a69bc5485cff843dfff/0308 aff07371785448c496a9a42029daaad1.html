Basic concepts

2 parameters:

Eps(float): Max radius of the neighbourhood of point P

MinPts(int): if the Eps-neighbourhood of P contains at least MinPts points in it, we call P a core object

Neighborhood of q

NEps(q)={P∈D ∣ dist(p,q)≤Eps}N_{Eps}(q)=\{P\in D\ |\ dist(p,q)\leq Eps\}NEps​(q)={P∈D ∣ dist(p,q)≤Eps}﻿

A point P is directly density-reachable from a point q under Eps, MinPts if…

P∈NEps(q)P \in N_{Eps}(q)P∈NEps​(q)﻿ // P is q’s neighbourhood

∣NEps(q)∣≥MinPts|N_{Eps}(q)|\geq MinPts∣NEps​(q)∣≥MinPts﻿ // q is a core object (A core project attracts points around it)

P2 is also directly density reachable from q

Merge all the points in NEsp(q) & NEsp(P2)N_{Esp}(q)\ \&\ N_{Esp}(P_2)NEsp​(q) & NEsp​(P2​)﻿ together

P is density-connected to a point q

If there is a point O such that both p & q are density-reachable from O

A density-based algo(DBScan)

A cluster is defined as a maximum set of density-connected points

Steps:

Arbitrarily select a data point P

Retrieve all points that are density-reachable from P under Eps & MinPts

Attract all points that are density-connected to P under Eps & MinPts

Repeat until all points in R(?) data set are processed

P is a core object!
